Metz is a powerful tool for creating interactive diagrams from TypeScript code. You can use it to express complex engineering ideas or even abstract ones. With metz, communication is just coding problem!

## Key Features

1. **Visual Runtime**: Metz includes its own visual runtime that executes the code you write, providing a dynamic and interactive experience.
2. **Story-Based Design**: The framework introduces the concept of "Stories," allowing you to segment your design and showcase different aspects of your system.
3. **Opinionated Framework**: Metz leverages battle-tested and mature ideas to provide a familiar and efficient `DX`.
4. **Browser native**: We run everything, from compilation to managing runtime, entirely in the browser.
